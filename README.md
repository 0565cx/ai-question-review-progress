# 各岗位 AI 审题进度

> 北森 · AI 面试官 · 专业能力 —— 各岗位题目审核（审题）协作仓库
>
> 本仓库参照企业微信共享表格「各岗位AI审题进度 → 8月迭代」的结构搭建，目的是让大家**像在共享表格里挂附件一样，在这里上传和下载题目/审核 Excel 文件**。

---

## 一、这是用来干什么的

在原来的共享表格里，每个岗位是一行，每个审核环节（待审核题表、一审、二审、三审……）的单元格里挂着对应的 Excel 题目文件。大家在表格里上传、下载、更新这些文件来推进审题。

本仓库把这种协作方式搬到 GitHub 上：

- **文件夹 = 共享表格里的"岗位 + 审核环节"**
- **文件 = 原来挂在单元格里的那个 Excel 题目表**
- 上传文件 = 在表格单元格里加附件
- 下载文件 = 从单元格里下载附件

不需要会用 Git 命令，**全程在 GitHub 网页上点鼠标即可**（见下方第四节）。

## 三、文件命名建议

沿用共享表格里的命名习惯，方便辨认：

```
<子行业>-<岗位>题目_待N轮审核.xlsx
```

示例：

- `汽车整车与零部件制造业-质量工程师题目_待1轮审核.xlsx`
- `半导体与集成电路制造业-设备工程师题目_待3轮审核.xlsx`
- `消费电子通信智能设备-自动化工程师_待一轮审核.xlsx`

如果是审核后的产出，也可以加后缀，例如 `_可保留的题目`、`_需要重新出题`、`_重新出答案`。

## 四、怎么上传 / 下载（网页操作，零基础）

### 上传文件

1. 在 GitHub 网页打开本仓库，点进你要上传的那个文件夹（例如 `校园招聘/制造业/新能源制造业/质量工程师/01_待审核题表/`）。
2. 右上角点 **Add file → Upload files**。
3. 把 Excel 文件拖进去，或点 **choose your files** 选择。
4. 下方填一句改动说明（例如"上传新能源-质量工程师待一轮审核题目"）。
5. 点绿色按钮 **Commit changes** 即完成上传。

### 下载文件

- **下载单个文件**：点进文件 → 右上角 **Download raw file**。
- **下载整个仓库**：仓库首页 **Code → Download ZIP**。

### 想新建岗位 / 行业文件夹

GitHub 网页不能直接建空文件夹，但可以**在上传文件时顺便建**：上传时在文件名最前面输入 `新文件夹名/`，例如 `互联网/产品经理岗/产品经理/01_待审核题表/题目.xlsx`，GitHub 会自动创建这些文件夹。

## 五、协作约定（建议）

- 文件名清楚标注「子行业-岗位-轮次」，不要用"最终版2_改改改.xlsx"这类名字。
- 同一份文件更新时，**用同名覆盖上传**，GitHub 会自动保留历史版本（可在文件页点 **History** 查看/回退）。
- 每次上传时在 commit 说明里写清楚干了什么，方便别人了解进度。
- 需要协作权限的同事，请仓库管理员在 **Settings → Collaborators** 里邀请。

## 六、与原共享表格的对应关系

| 共享表格 | 本仓库 |
| --- | --- |
| 一个岗位行 | 一个岗位文件夹 |
| 审核环节列（待审核题表 / 各轮审核 / 汇总） | 岗位文件夹下的 `01~06` 阶段子文件夹 |
| 单元格里的附件 | 文件夹里的 Excel 文件 |
| 改动记录 | Git 提交历史（History） |

> 本仓库结构来源：企业微信共享表格「各岗位AI审题进度」中的 **8月迭代** sheet。如需扩展到 6月/4月迭代或更多行业岗位，按相同层级新建文件夹即可。
